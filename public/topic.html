Introduction to HTML and CSS

HTML and CSS are the two fundamental technologies used to build web pages. HTML provides the structure, while CSS defines the visual appearance.

HTML

HTML (HyperText Markup Language) uses elements and tags to describe the content of a webpage, such as headings, paragraphs, images, and links.

CSS

CSS (Cascading Style Sheets) is used to style HTML elements. It controls colours, fonts, spacing, layout, and the overall design of the website.

Why they matter

Together, HTML and CSS form the foundation of web development. Every website uses them.
